El token se guarda cifrado. Al editar una cuenta, deja el token vacío para conservar el actual. Necesitas un token de System User con permiso ads_read (Business Manager).

La sincronización corre en segundo plano: el botón vuelve enseguida y la «Última sync» se actualiza al terminar (recarga para ver). Puedes elegir hasta «Todo (~3 años)». La descarga se hace por ventanas (de viejo a nuevo), así que no falla por «reduce the amount of data».

Para automatizar, programa un cron cada 15 min: python manage.py sincronizar_ads --dias 3 (la primera carga histórica también puedes lanzarla aquí con «Todo», o por comando: sincronizar_ads --dias 1095).
